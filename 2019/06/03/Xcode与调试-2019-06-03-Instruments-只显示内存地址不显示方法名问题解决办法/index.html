Instruments-只显示内存地址不显示方法名问题解决办法

发表于 2019-06-03

  1. 需要工程生成 dSYM 文件，Instruments 才能正常地根据内存地址找到方法名。
    默认情况下，debug 模式是不生成 dSYM 文件的。稍作修改即可。

  2. Instruments 寻找 dSYM 文件，依赖的是系统的 Spotlight，如果将 dSYM 文件所在的路径排除在了 Spotlight 索引之外，那么 Instruments 也是不能寻找到的。
    针对这中原因引起的问题，有两个解决办法：

    1. 手动设置 dSYM 文件的位置
      先运行 Instruments ，正常捕获一下程序，再点击停止。选择 File -> Symbols…

      在对应的窗口中，手动设置 dSYM 文件的位置，重新运行即可。
    2. 设置Instruments 搜索 dSYM 文件的路径
