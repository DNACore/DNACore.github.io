Windows 安装 node.js 绿色版

  1. 下载好绿色版压缩包
    https://nodejs.org/en/download/

  2. 将压缩包所有文件解压到某个文件夹内

  3. 在系统属性的环境变量里填入上一步解压到的文件夹路径。

  4. 直接打开 cmd 输入 npm 看看是否有响应。

与绿色版 node.js 有关的路径和文件

  • 缓存：
    %APPDATA%\npm-cache\

  • 配置文件路径
    "C:\Users\[用户名]\.npmrc"

npm使用

  • 设置下载镜像

  1. 通过命令设置
    npm config set registry https://registry.npm.taobao.org

  2. 编辑 .npmrc
    registry = https://registry.npm.taobao.org

npm 更新

  • 更新 npm 到最新版本
    npm i npm@latest -g
    或者
    npm install -g npm

  • 更新 npm 到指定版本
    npm -g install npm@5.6.0

  • npm 源管理工具 nrm

  • nrm ls 列出源

  • nrm use XXX 使用源

  • npm install --cache /tmp/empty-cache 指定临时缓存目录
