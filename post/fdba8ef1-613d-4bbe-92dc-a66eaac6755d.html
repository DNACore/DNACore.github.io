macOS-快捷键

↵⇥→←↓↑⎋⇧®©⌫⌃⌥⌘

macOS系统

  • 最小化窗口 ⌘ + m
  • 关闭权限、提醒类弹窗 ⌘ + ⌥ + ⇧ + esc

Finder 访达快捷键

  • 在新窗口中打开上一级文件夹 ⌘ + control + 上

  • 在当前窗口中打开上一级文件夹 ⌘ + option + 上

  • 打开用户文件夹⌘+Shift+H

  • 快速文件夹跳转⌘+Shift+G

  • 快速打开实用工具文件夹⌘+Shift+U

  • command+左击 Dock 栏程序：打开对应程序所在的位置。

  • 查看选中文件信息：option + i

  • 批量查看选中文件信息：⌘ + option + i

  • 显示隐藏文件：⌘+shift+.
    另外的方法
    显示：defaults write com.apple.finder AppleShowAllFiles -bool true
    隐藏：defaults write com.apple.finder AppleShowAllFiles -bool false

  • 显示路径：⌘+option+p

  • 复制路径：⌘+option+c如果被复制的路径包含软链接，那么复制出来的路径是原始路径。

  • 移动文件：复制之后 -> ⌘+Shift+v

  • 打开macOS驱动器列表 ⌘+Shift+c

  • 浏览上一层文件夹 ⌘+[

  • 浏览下一层文件夹 ⌘+]

  • 删除项目到废纸篓 ⌘+Delete

  • 删除项目 option+⌘+Delete

  • 清空废纸篓 Shift+⌘+Delete

  • 清空废纸篓不确认 option+Shift+⌘+Delete

  • 截图 触控栏 保存到桌面 ⌘+Shift+G

  • 截图 触控栏 保存到剪贴板 ⌘+Shift+control+G

其它

  • Finder偏好设置：⌘+,

Safari

  • 强制阅读模式 Shift+⌘+R
