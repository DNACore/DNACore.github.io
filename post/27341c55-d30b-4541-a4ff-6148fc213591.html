macOS 文件路径

钥匙串访问

系统路径

  • 第三方偏好设置路径

    • 个人~/Library/PreferencePanes
    • 系统/Library/PreferencePanes

  • 系统内置偏好设置路径
    /System/Library/PreferencePanes

  • 系统更新下载位置
    /Library/Updates

  • 软件默认设置路径
    ~/Library/Preferences

  • 特权文件路径，删除后可取消某些app的特权

    更多关于帮助程序的内容请看帮助程序是什么

  • 拼音输入法用户词库自造词位置

  • icns 图标位置

  • 铃声路径
    /System/Library/PrivateFrameworks/ToneLibrary.framework/Versions/A/Resources/Ringtones点击直达
    设置里面的铃声/System/Library/Sounds点击直达
    /Library/Audio/Apple Loops[点击直达](file:///Library/Audio/Apple Loops)

  • 桌面壁纸
    /Library/Desktop Pictures/
    [点击直达](file:///Library/Desktop Pictures/)
    [点击直达localhost形式](file://localhost/Library/Desktop Pictures/)

  • 部分系统程序 归档实用工具、文件夹操作设置、访达、聚焦、抹掉助理等等
    /System/Library/CoreServices
    点击直达

  • 鼠标指针
    /System/Library/Frameworks/ApplicationServices.framework/Versions/A/Frameworks/HiServices.framework/Versions/A/Resources/
    点击直达

  • 自动操作素材库路径 Automator
    /System/Library/Automator/

  • 用户添加的自动操作素材库路径 Automator
    ~/Library/Automator

  • 快速操作存储路径
    ~/Library/Services

  • 系统快速操作存储路径 /System/Library/Services
    “快速操作”会被显示到右键菜单——服务中
    系统设置——键盘——快捷键——服务中也可以编辑快速操作列表显示的内容

  • iCloud云盘 iCloud Drive
    ~/Library/Mobile\ Documents/com~apple~CloudDocs/

  • 新用户目录模板
    /System/Library/Templates

应用软件路径

  • 预览图片查看器中自定义颜色存储路径
    ~/Library/Colors里面的clr文件

  • Telegram 升级文件路径
    ~/Library/Caches/ru.keepcoder.Telegram/org.sparkle-project.Sparkle/PersistentDownloads

  • 语音备忘录 Voice Memo
    $HOME/Library/Application Support/com.apple.voicememos/Recordings
    https://apple.stackexchange.com/questions/340647/where-are-the-audio-files-of-voice-memos-stored-on-macos-mojave/340662

  • QQ 数据文件夹

    • ~/Library/Containers/com.tencent.qq/
    • 图片路径~/Library/Containers/com.tencent.qq/Data/Library/Caches/Images

  • 企业微信、微信企业版升级包路径
    ~/Library/Containers/com.tencent.WeWorkMac/Data/Upgrade

  • 微信路径

    微信手机聊天记录备份路径，微信聊天记录备份是增量备份
    ~/Library/Containers/com.tencent.xinWeChat/Data/Library/Application Support/com.tencent.xinWeChat/2.0b4.0.9/Backup

  • Source Tree 更新文件下载路径
    ~/Library/Caches/com.torusknot.SourceTreeNotMAS/org.sparkle-project.Sparkle/PersistentDownloads
    点击直达

  • 百度网盘数据库路径
    ~/Library/Application Support/com.baidu.BaiduNetdisk-mac

  • Charles 大文件缓存路径
    $TMPDIR/charles8637968132878592540.tmp

  • Charles 配置文件路径
    ~/Library/Application Support/Charles
    $HOME/Library/Preferences/com.xk72.charles.config

  • Mac App Store 下载软件保存路径
    $(getconf DARWIN_USER_TEMP_DIR)/com.apple.appstoreagent/com.apple.appstore/

  • macOS 备份的iOS设备文件路径
    ~/Library/Application Support/MobileSync/Backup

  • Instruments 用户模板保存位置
    ~/Library/Application Support/Instruments/Templates

  • VS Code 用户自建代码片段
    ~/Library/Application Support/Code/User/snippets/

垃圾文件路径

  • /private/var/db/diagnostics/Signpost
