如何完全重新初始化远端git仓库

有时候我们想重新初始化远端的git仓库，比如说完全重新初始化GitHub上面的仓库。

可以按照如下方法做：

  1. cd <git_dir> //切换到你的本地git仓库目录
  2. rm -rf .git //删除本地的.git目录
  3. git init //重新创建本地的.git目录
  4. 如果并不想提交任何内容，可以直接执行git commit -m 'Initial commit' --allow-empty；如果有需要第一次提交的内容，看下面步骤：
  5. git add . 将当前目录下所有的文件添加到git中
  6. git commit -m 'Initial commit' 创建第一个提交
  7. git remote add origin <url>
  8. git push --force --set-upstream origin master
