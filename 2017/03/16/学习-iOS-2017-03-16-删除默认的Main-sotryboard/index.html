删除默认的 Main.sotryboard

  1. 我们先删除 Xcode 自动为我们建立的 Man.storyboard 文件和两个 ViewController 类文件。
  2. 在工程的 Target – General – Deployment Info – Main interface 里面，将其值改为空。
  3. 建立新的默认视图
  4. 在 AppDelegate 里面添加代码，程序启动时将我们添加的视图设置为 UIWindow 的根视图。
