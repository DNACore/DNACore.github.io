打包过程遇到的问题以及解决方案

发表于 2019-03-26

  • Duplicated module name: XXX
    此问题一般出现在有自有 js module 包的时候，并且使用了npm install来安装包。
    npm有时并不能正确处理包之间的依赖关系。此时用yarn install来安装包即可。

  • node版本问题
    react native 工程依赖于 node 。并且不同版本 RN 工程对 node 的版本要求不同。
    当出现node版本过低的错误时，可对本机安装的 node 进行版本升级。
