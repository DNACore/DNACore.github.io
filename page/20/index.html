WES7激活

发表于 2018-11-28 更新于 2019-09-11

Windows Embedded 7 嵌入版 激活

WES7 激活序列号

需要在安装时输入这个序列号或者用ICS植入序列号（此密钥为永久激活密钥）
XGY72-BRBBT-FF8MH-2GG8H-W7KCW

用命令安装密钥（此密钥为永久激活密钥）

  • 其他激活序列号

Windows Embedded Standard 7 Runtime (x86) - DVD (English)
产品密钥：MPMVY-PP762-WWVBC-83RXJ-2H7RH 静态激活密钥

Windows Embedded Standard 7 Runtime (x64) - DVD (English)
产品密钥：MPMVY-PP762-WWVBC-83RXJ-2H7RH 静态激活密钥

Windows Embedded Standard 7 Toolkit (x86) - DVD (English)
产品密钥：7XT84-3XQFP-YMXQF-PP96H-F74DV 静态激活密钥

WES7官定免费使用180天，到期后如果需要再用下去，并不需要重新安装，只需要按照微软公布的正规方法：执行一下slmgr.vbs /rearm重置激活，就可以再使用180天，可以这样重置激活6次，这就等于可以免费使用3年。3年后，按照微软官网公布的方法，将注册表项：HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion\SoftwareProtectionPlatform\"SkipRearm"=dword:00000000的值更改为1，又可以重置激活，可以这样重复 8 次，因此，官方WES7可以免费使用 7 年。当然，重新安装一下，又可以使用7年。

参考：
http://bbs.pcbeta.com/viewthread-835562-1-1.html
http://bbs.pcbeta.com/viewthread-835562-1-1.html

开发相关路径

  • Xcode Developer Image
    /Applications/Xcode.app/Contents/Developer/Platforms/iPhoneOS.platform/DeviceSupport

  • Provision Profile 描述文件路径
    ~/Library/MobileDevice/Provisioning\ Profiles/

  • 模拟器路径
    ~/Library/Developer/CoreSimulator/Devices/

  • 模拟器 KeyChain 数据库文件路径
    ~/Library/Developer/CoreSimulator/Devices/{SimilatoeUUID}/data/Library/Keychains/keychain-2-debug.db

  • Xcode 缓存、配置、编译结果文件路径
    此路径下的内容包括 Archives、DerivedData、DocumentationCache、iOS Device Logs、iOS DeviceSupport、UserData

    ~/Library/Developer/Xcode/

  • SDK 路径
    /Applications/Xcode.app/Contents/Developer/Platforms/iPhoneOS.platform/Developer/SDKs

  • Xcode Code Snippet 路径
    ~/Library/Developer/Xcode/UserData/CodeSnippets

WES 7 - Windows Embedded Standard 7 SP1 安装

发表于 2018-11-09 更新于 2019-09-10

WES 7

WES 7 安装

https://lvtao.net/server/windows7-wes7.html

WES 7 安装 MTP 驱动

  1. 从原始安装镜像中提取WinEmb-INF-wpdmtp.cab程序包，放入MTPDriver文件夹中。
  2. 以管理员身份打开 cmd 命令行，切换到MTPDriver文件夹。
  3. 执行DISM /online /add-package /packagepath:.，命令行最后的点号表示需要添加的程序包就在当前目录下。
  4. 待程序执行完成之后重启电脑即可。
