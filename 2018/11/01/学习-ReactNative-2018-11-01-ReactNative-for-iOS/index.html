添加各种第三方库

  • 添加 Lottie
    项目地址：https://github.com/react-community/lottie-react-native
    执行命令：

    执行完成之后在General中的Embedded Binaried中添加Lottie.framework

  • 添加 PushNotificationIOS
    参考：https://facebook.github.io/react-native/docs/pushnotificationios.html#content

  1. Add the following to your Project: node_modules/react-native/Libraries/PushNotificationIOS/RCTPushNotification.xcodeproj

  2. Add the following to Link Binary With Libraries: libRCTPushNotification.a

  3. Build Phases 中必须添加的第三方库

    libRNDefaultPreference.a
    libRNFetchBlob.a
    libBVLinearGradient.a
    libRNDeviceInfo.a
    libLottieReactNative.a
    libLottie.a

针对项目中出现的问题

  1. 需要删除工程中 SDWebImage 多余的文件引用。

  2. Header Search Paths 添加`”$(SRCROOT)/RARTools/DevelopPack/RDPImage/Vendor/SDWebImage/SDWebImage”

  3. 缺少 Lottie.h SDWebImage.h 的头文件
    Build Phases -> 添加 -> 重命名为 Embed Frameworks -> 添加 Lottie.framework 和 SDWebImage.framework

  4. Cycle in dependencies between targets 'Lottie_iOS' and 'LottieReactNative'
    解决：https://github.com/react-community/lottie-react-native/issues/385

  5. Error: Couldn't find preset "module:metro-react-native-babel-preset"
    解决办法：

    1. npm i metro-react-native-babel-preset --save-dev
    2. 修改.babelrc文件
      将其中的"presets": ["module:metro-react-native-babel-preset"]修改为"presets": ["react-native"]
