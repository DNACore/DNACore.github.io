浏览器配置about:config

  • Firefox防止WebRTC泄漏IP
  1. about:config
  2. media.peerconnection.enabled设置为false

参考：https://www.ivpn.net/knowledgebase/158/My-IP-is-being-leaked-by-WebRTC-How-do-I-disable-it.html

  • 延长自动保存会话时间，减少磁盘写入
    about:config -> browser.sessionstore.interval -> 900000
    参考：https://www.servethehome.com/firefox-is-eating-your-ssd-here-is-how-to-fix-it/

  • Firefox 证书管理
    https://support.mozilla.org/zh-CN/kb/setting-certificate-authorities-firefox
    安装某些VPN软件后，有可能会修改Firefox设置about:config中enterprise_roots有关的选项
    https://support.mozilla.org/en-US/questions/1230877

  • 历史记录保存天数
    browser.migrate.chrome.history.maxAgeInDays

Firefox compact mode 紧凑模式

  1. 在地址栏里输入about:config，然后按回车。
  2. 有时会出现警告页面。点击我接受此风险，请继续！接受风险并继续，以打开 about:config 页面。
  3. 找到首选项 browser.compactmode.show。
  4. 将其值切换为 true，然后关闭该面板。
  5. 点击菜单按钮Fx89menuButton打开菜单面板。
  6. 点击 更多工具。
  7. 选择 定制工具栏...。
  8. 在面板底部，点击密度。
  9. 在菜单选项中选择 紧凑（已经不支持）。

Firefox container

  • 当点击了【Remember my decision for this site】发现没有办法修改回去怎么办？
    1. 打开about:debugging#/runtime/this-firefox
    2. 找到Firefox Multi-Account Containers，点击右侧的Inspect
    3. 定位到Storage/Extension Storage这里就可以直接修改扩展的配置，找到对应的网址然后修改即可。
