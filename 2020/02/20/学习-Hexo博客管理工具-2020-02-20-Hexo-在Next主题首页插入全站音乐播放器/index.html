前言

APlayer是一个很好用的Hexo博客音乐播放器，可是网上的教程都是在讲如何用APlayer将音乐插入到某个单独的文章中。
这篇文章就讲讲怎么将APlaye播放器插入到Hexo博客的首页，作为全站可用的播放器。

添加 APlayer 代码到主题资源中

  1. 从 GitHub 上下载 APlayer 代码
    移步至此，https://github.com/MoePlayer/APlayer
    点击Clone or download，在弹出的菜单中选择Download ZIP。
    选择好下载位置。

  2. 解压刚刚下载到的APlayer-master.zip文件。将里面的dist文件夹放到主题目录中source目录下，例如：themes/next/source/。
